# ParkAlert
Framework: [Nest](https://github.com/nestjs/nest) 

## Installation

```bash
$ pnpm i
```

## Running the app

```bash
# development
$ pnpm run dev

# production mode
$ pnpm run dev:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
